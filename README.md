# Item Catalog Frontend

## Overview
This is the frontend of the Item Catalog project — a simple and responsive web application built using the MERN stack.  
The web application allows users to browse, view, and manage products, add items to the cart, and view detailed information for each product. Also this application allows Admin to do the CRUD operations to add,get,update,delete products.

---

## Features
- Product listing with name, image, price, and category, stock, size  
- View detailed information for each item  
- Add and remove items from the cart  
- Dynamic cart updates with quantity control  
- User authentication (Sign in / Sign up)  
- Fully responsive design for desktop and mobile  
- Clean UI using Bootstrap and custom CSS  and tailwind CSS
- Admin CRUD operations

---

## Tech Stack
**Frontend:**  
- React (Vite)  
- React Router DOM  
- Axios  
- Bootstrap  
- Context API (for state management)  

**Backend:**  
The backend for this project is built using Node.js, Express, and MongoDB.
